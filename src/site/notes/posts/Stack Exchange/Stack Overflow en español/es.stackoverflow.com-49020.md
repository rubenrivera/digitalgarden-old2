# Alineación vertical central con CSS

# Pregunta
Tengo una etiqueta div que contiene text. Quireo que el contenido de este dive se alinee verticalmente al centro. ¿Cómo puedo lograrlo?

## Fragmento
<!-- begin snippet: js hide: false console: false babel: false -->

<!-- language: lang-css -->

    #box
    {
      height: 180px;
      width: 400px;
      background: #000;
      font-size: 48px;
      font-style: oblique;
      color: #FFF;
      text-align: center;
      margin-top: 20px;
      margin-left: 5px;
    }

<!-- language: lang-html -->

    <div id="box">
      Lorem ipsum dolor.
    </div>

<!-- end snippet -->
